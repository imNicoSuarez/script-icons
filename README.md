Script Generate Icons
=====


## Install IMAGEMAGIC

http://www.imagemagick.org/script/binary-releases.php

## Install NODE.JS

http://nodejs.org/

##Generate Icon
~~~bash
  node icons.js ----max-old-space-size=6000
  sh iconScript.sh
~~~

## Generate Line
~~~bash
  node lines.js
  sh lineScript.sh
~~~

## Generate Polygon
~~~bash
  node polygons.js
  sh polygonScript.sh
~~~
